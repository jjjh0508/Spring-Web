표현식4 - 선택 변수 표현식 *{...}

html 출력 th:text, th:utext, th:value

el 태그의 값을 태그 내부의 값으로 작성하기 위해서는 th:text 또는 th:utext를 사용할 수 있다. th:text는 escape가 적용되어 태그를 단순한 문자열로 처리하지만 utext는 escape가 적용되지 않아 태그를 태그로써 인식 할 수 있다. (DB에 태그가 포함된 무자열을 저장하는 경우) th:value는 태그의 value 값을 지정할 수 있다.

표현식5 - 인라인 표현식[[...]], [[(...)]

변수 표현식의 값을 html에서 직접 표시하기 위해서 th:text와 같은 [[...]]를 사용하여 th:utext와 같은 [(...)]을 사용할 수 있다. 대괄호 묶어 이와 같이 변수 표현식의 값을 가져오는 것을 인라인 모드라고 하며 인라인 모드는 text 모드와 자바스크립트 모드가 있다. 우리는 변수 표현식의 값을 자바스크립트에서 쓰는 것이 아닌 html에서 사용하려는 것이므로 th:inline="text"를 태그에 속성 값으로 주고 써야하지만 th:inline="text"를 쓰지 않아도 적용된다. 반면 인라인 모드를 적용하지 않으려면 th:inline="none"을 속성값으로 주면 변수 표현식의 값이 인라인모드로 사용하지 않아 ([[]]) 또는 [()]를 인식하지 않음) 단순 문자열로 처리할 수 있다. 자바스크립트에서 사용하려면 th:inline="javascript" 를 태그에 속성값으로 주고 써야하지만 역시나 th:inline=javascript"를 쓰지 않아도 됨

  • [[${hello}]]
  • [(${hello})]
  • [[${hello}]]
  • [(${hello})]

th:block

범위를 지정하고 있을 때 사용한다. th:block을 통해 해당 범위에 변수나 객체를 적용하거나 조건에 해당되는지에 따라 해당범위를 보여주거나 보여주지 않을 때 사용 할 수 있다.
